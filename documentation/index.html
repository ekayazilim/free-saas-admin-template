EkaAdmin Documentation

Complete guide to setting up, customizing, and extending the EkaAdmin Premium SaaS Admin Template.

Introduction

EkaAdmin is a premium HTML5 admin dashboard template with 129+ pages, 20+ widgets, and 5 dashboard variants. Built with vanilla HTML, CSS, and JavaScript — no frameworks or build tools required.

This template is designed for developers who want a quick start for admin panels, SaaS dashboards, CRM systems, and e-commerce backends.

Key Features

129+ HTML Pages

Dashboards, e-commerce, authentication, components, forms, tables, charts, and more.

Dark & Light Mode

One-click theme toggle with localStorage persistence across page reloads.

Multiple Layouts

Vertical sidebar, horizontal menu, boxed, and full-width layout options.

Theme Customizer

Live settings panel for colors, layout, sidebar style, and header options.

Installation

Quick Start

  1. Extract the downloaded ZIP file to your web server directory.
  2. Open index.html in your browser — or point your domain to the extracted folder.
  3. Navigate to auth/login.html to access the admin dashboard.
  4. Use the default demo credentials: admin@ekadmin.com / password123
No build tools, npm, or package managers required. Simply extract and open in a browser.

Dark Mode

Dark mode is controlled via the data-theme attribute on the <html> element. All dark mode styles are in eka-dark.css.

<!-- Light mode -->
<html lang="en" data-theme="light">

<!-- Dark mode -->
<html lang="en" data-theme="dark">

The theme preference is automatically saved to localStorage and persists across page reloads.

Layouts

Vertical (Default)

The default layout uses a left sidebar with collapsible navigation. The sidebar automatically collapses on smaller screens.

Horizontal

The horizontal layout places the navigation in the topbar area. Activate by adding data-layout="horizontal" to the HTML element.

Colors

All colors are defined as CSS custom properties in eka-variables.css:

:root {
  --eka-primary: #405189;
  --eka-secondary: #3577f1;
  --eka-success: #0ab39c;
  --eka-warning: #f7b84b;
  --eka-danger: #f06548;
  --eka-info: #299cdb;
}

To change the color scheme globally, simply update these values in the variables file.

Charts (ApexCharts)

All charts are powered by ApexCharts. Include the library and initialize:

<script src="https://cdn.jsdelivr.net/npm/apexcharts"></script>
<div id="myChart"></div>
<script>
new ApexCharts(document.querySelector('#myChart'), {
  chart: { type: 'area', height: 350 },
  series: [{ name: 'Revenue', data: [31, 40, 28, 51, 42] }],
  colors: ['#405189']
}).render();
</script>
