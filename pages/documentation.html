Introduction

EkaAdmin is a premium, production-ready admin dashboard template built with vanilla HTML, CSS, and JavaScript. It features a comprehensive set of pages, components, and utilities designed for building modern web applications.

EkaAdmin requires no build tools or package managers. Simply open the HTML files in your browser.
Features
  • 90+ Premium Pages
  • 5 Dashboard Variants
  • 11 Application Modules
  • Dark/Light Mode
  • ApexCharts Integration
  • Responsive Design
  • Lordicon Animations
  • CKEditor 5 Support

Installation

Getting started with EkaAdmin is simple:

git clone https://github.com/ekayazilim/ekaadmin.git
cd ekaadmin
open dashboards/index.html

Or simply extract the downloaded ZIP file and open dashboards/index.html in your browser.
