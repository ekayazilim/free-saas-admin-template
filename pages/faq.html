Frequently Asked Questions

Find answers to common questions about EkaAdmin

What is EkaAdmin?
EkaAdmin is a premium SaaS admin dashboard template built with HTML, CSS, and JavaScript. It includes 80+ pages, multiple dashboard variants, and a comprehensive component library.
How do I customize the theme?
Use the Theme Settings panel on the right side to switch between light/dark mode, change sidebar colors, and adjust the layout. All settings are saved in localStorage.
Is it responsive?
Yes, EkaAdmin is fully responsive across desktop, tablet, and mobile devices. The sidebar collapses automatically on smaller screens.
What libraries are used?
We use ApexCharts, Font Awesome, Lordicon, Flatpickr, DataTables, CKEditor 5, SortableJS, FullCalendar, and Choices.js via CDN.
